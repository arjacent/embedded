# Traffic Light Controller
My implementation of Lab 10 of Embedded Systems Change the World from Texas A&M.

![Lab10image001.png](Lab10image001.png)

## Requirements

### Software
* Windows 7 or higher
* Keil uVision4 IDE

### Hardware
* TI Tiva C Series TM4C123GXL
* Breadboard + wires
* 3x Switches
* 3x 10kΩ resistors
* 6x LEDs (green/yellow/red)
* 6x 470Ω resistors
